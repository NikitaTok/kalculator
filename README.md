# Калькулятор на Python с Tkinter
Расширенный графический калькулятор с функциями, поддержкой памяти и некоторых клавиш.

### Основные операции
- Базовые арифметические операции (+, -, ×, ÷)
- Поддержка скобок и приоритета операций
- Работа с десятичными дробями

### Функции
- Тригонометрия (sin, cos, tan) в градусах
- Квадрат и квадратный корень
- Логарифмы (log₁₀ и натуральный ln)
- Константы π и e
- Факториал
- Проценты и смена знака

### Память
- M+ — добавление к памяти
- MR — восстановление из памяти
- MC — очистка памяти

### Управление
- Клики мышью по кнопкам
- Поддержка клавиатуры
- Backspace для удаления символа
- Esc для полной очистки

## Установка и запуск

### Требования
- Python 3.8 или выше
- Стандартная библиотека Python (Tkinter, math)

### Запуск
# Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/kalculator.git
cd kalculator

# Запустите калькулятор
python kalculator.py
